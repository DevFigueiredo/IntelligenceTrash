
## Sobre o Projeto
Resumo:
- Instalar o PHP e extensões do Laravel
- Instalar o Composer
- Baixar as dependências do projeto: composer update
- Rodar no PowerShell: php artisan serve
- Rodar o Banco de Dados: php artisan migrate
